Open-Source DICOM viewers

  • OsiriX: OsiriX is a popular open-source DICOM viewer for MacOS. It offers advanced features such as 3D rendering, multi-planar reconstruction, and image fusion.
  • Slicer: Slicer is an open-source platform for medical image informatics, visualization, and analysis. It provides a range of tools for processing and analyzing medical images, including DICOM viewers.
  • ImageJ: ImageJ is a Java-based open-source image processing and analysis software. It provides plugins for viewing and analyzing DICOM images, as well as tools for image processing and manipulation.
  • MedINRIA: MedINRIA is an open-source software package for medical image processing and visualization. It offers features such as 3D rendering, diffusion tensor imaging, and fMRI analysis.

Commercial DICOM viewers

  • OsiriX MD: OsiriX MD is a commercial version of OsiriX that offers advanced features such as FDA-approved diagnostic tools, support for medical devices, and integration with PACS systems.
  • RadiAnt: RadiAnt is a commercial DICOM viewer for Windows. It offers advanced features such as multi-touch support, customizable workspaces, and a built-in DICOM server.
  • Aeskulap: Aeskulap is a commercial DICOM viewer for Linux. It offers features such as 3D rendering, multi-planar reconstruction, and image fusion.
  • ClearCanvas: ClearCanvas is a commercial DICOM viewer for Windows. It offers features such as customizable workspaces, multi-monitor support, and integration with PACS systems.

Modalities

  • X-rays: X-ray images are among the most common types of medical images. DICOM supports both 2D and 3D X-ray images, including digital radiography and computed tomography (CT) scans.
  • CT scans: CT scans use X-rays to create 3D images of the body. DICOM supports CT images with varying slice thicknesses, allowing for the reconstruction of high-resolution 3D images.
  • MRIs: Magnetic Resonance Imaging (MRI) uses magnetic fields and radio waves to create images of the body's internal structures. DICOM supports a range of MRI image modalities, including T1, T2, and diffusion-weighted imaging.
  • Ultrasounds: Ultrasound images use high-frequency sound waves to create images of the body's internal structures. DICOM supports a range of ultrasound image modalities, including 2D, 3D, and Doppler images.
  • PET scans: Positron Emission Tomography (PET) scans use radioactive tracers to create 3D images of the body's internal structures. DICOM supports PET images with varying slice thicknesses, allowing for the reconstruction of high-resolution 3D images.
  • SPECT scans: Single-Photon Emission Computed Tomography (SPECT) scans use radioactive tracers to create 3D images of the body's internal structures. DICOM supports SPECT images with varying slice thicknesses, allowing for the reconstruction of high-resolution 3D images.
  • Mammography: Mammography is a type of X-ray used to screen for breast cancer. DICOM supports mammography images with high-resolution and high-contrast imaging capabilities.
  • Endoscopy: Endoscopy involves the use of a camera to view the inside of the body's organs. DICOM supports endoscopy images with various modalities, including upper and lower gastrointestinal endoscopy, bronchoscopy, and cystoscopy.
